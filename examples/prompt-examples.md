# Prompt examples

## Set a default type

```text
Use the mbti-agent skill with INTJ as my default coding collaboration style.
```

```text
For this repository, use ENFJ mode unless I override it.
```

## Override for one task

```text
Switch to ISTP mode for this debugging session. Keep explanations short and focus on runtime evidence.
```

```text
Use ENTP mode for the architecture brainstorm, then converge on one recommendation.
```

## Use overlays

```text
I'm stuck. Use INTP mode, but give me one concrete next debugging step instead of several theories.
```

```text
I'm rushed. Use INFJ mode but keep the plan short and start implementing after the risk scan.
```

```text
I'm learning React. Use ESTP mode, but explain each debugging step as we go.
```

```text
I'm exploring options for a plugin system. Use ISTJ mode, but include one novel approach alongside the proven one.
```

```text
I'm fatigued. Use ENFP mode but keep this simple: one recommendation, short rationale, next step.
```

## Disable

```text
Ignore MBTI for this task and just follow the repository instructions.
```
